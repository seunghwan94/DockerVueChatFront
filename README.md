# DockerVueChatFront (vue.js)
챗봇 앞단으로 socket.io 를 이용하여 화면 구성

Back단이 있어야 사용가능

## npm setting
```
npm i
```

### .gitignore랑 같은 위치에 config.json 파일 생성

example

```
{
  "server": {
    "host": "",
    "port": 3000
  }
}

```

### 실행
```
npm run serve
```

### Back 단 바로가기
[DockerVueChatBack 바로가기](https://github.com/seunghwan94/DockerVueChatBack).
